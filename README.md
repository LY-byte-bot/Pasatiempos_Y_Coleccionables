# 🎨 El Rincón del Coleccionista

Sitio web dedicado a los pasatiempos y objetos coleccionables. Permite explorar diferentes categorías, conocer curiosidades, guardar artículos favoritos temporalmente y gestionar una colección personal desde una interfaz dinámica y responsiva.

## ✨ Características

- Catálogo de cómics, mangas, discos y pinturas.
- Secciones de antigüedades, autógrafos y cartas coleccionables.
- Contenido sobre máquinas arcade, rockolas, pinball y figuras de artistas.
- Buscadores dinámicos por nombre.
- Ordenamiento alfabético y por fecha.
- Botones para expandir y contraer información.
- Carrusel automático de contenido.
- Quiz para coleccionistas.
- Sección de preguntas frecuentes.
- Sistema temporal de favoritos.
- Eliminación de favoritos mediante confirmación.
- Menú hamburguesa para dispositivos móviles.
- Botón dinámico para volver al inicio.
- Diseño adaptable a computadoras, tabletas y celulares.

## 📦 Gestión de la colección

El proyecto incorpora un módulo para administrar objetos coleccionables durante la sesión actual.

Este módulo permite:

- Insertar nuevos coleccionables.
- Buscar por ID, nombre, categoría o estado.
- Editar información.
- Eliminar objetos mediante confirmación.
- Mostrar los registros en una tabla dinámica.
- Ordenar por nombre, precio, cantidad o categoría.
- Calcular subtotales, totales y promedios.
- Identificar el objeto de mayor precio.
- Identificar la categoría con más unidades.
- Generar una matriz de categorías y estados.

Los datos se almacenan utilizando `sessionStorage`. Permanecen disponibles durante la sesión actual, pero se eliminan al cerrar la pestaña o el navegador.
